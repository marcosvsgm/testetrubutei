# Sistema de Estoque - Frontend

Frontend em React para o sistema de gerenciamento de estoque.

## Tecnologias

- React 18
- Vite
- Axios

## Instalação

```bash
npm install
```

## Executar em modo desenvolvimento

```bash
npm run dev
```

O aplicativo estará disponível em `http://localhost:3000`

## Build para produção

```bash
npm run build
```

## Funcionalidades

- Gerenciamento de Categorias (CRUD completo)
- Gerenciamento de Produtos (CRUD completo)
- Interface responsiva e moderna
- Validação de formulários
- Mensagens de sucesso e erro

## API Backend

O frontend espera que a API esteja rodando em `http://localhost:8000/api`

Certifique-se de que o backend Laravel está em execução antes de iniciar o frontend.
